# 发送给doubao的针对图片描述prompt
请提炼这张图中可用于生成相似画面的中文描述，只输出描述本身，不要标题、分点、解释、评价、提问或其他额外内容。重点描述场景、主体、位置关系、前中后景、镜头视角、构图、背景、光线、色调、材质和整体氛围；如果有人物，可以写人物的穿着、动作和在画面中的位置，但不要描述人种、身份或做主观推测。用具体、自然、可还原的中文表达，避免“这是一张”“图片中”“可能”等措辞。
