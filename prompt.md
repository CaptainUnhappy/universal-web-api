主题:1 打造一个完整的工程车主题生日派对场景：
背景：砖墙 / 工地风背景布，搭配 “CAUTION” 警示条、路障装饰
前景：生日蛋糕（工程车造型）、派对桌布、餐具，气球套装作为主装饰，数字气球放在 C 位
点缀：小轮胎装饰、迷你工程车玩具，和气球主题呼应
光线用暖光，营造温馨热闹的派对氛围，整体画面明亮、色彩饱满
2 拍摄孩子和气球互动的画面：
孩子戴着安全帽，拿着气球，对着镜头笑 / 和气球玩耍
也可以拍孩子抱着气球、坐在气球旁边玩玩具的画面，突出 “孩子和气球互动很开心”
背景简化，重点突出孩子和气球的互动，捕捉孩子真实的笑容和情绪
3 室内小型生日会，气球作为背景墙装饰，搭配蛋糕和礼物
场景 2：户外草坪派对，气球挂在帐篷上，搭配野餐垫和小玩具
场景 3：数字气球换成 “3/4/5”，展示不同年龄的适配性
光线与画质：所有场景图都要 8K 超高清，色彩还原真实，避免过曝或偏色，让产品的黄 / 红配色更鲜艳吸睛
背景选择：优先用干净、简约的背景，避免复杂元素抢了气球的风头，重点突出产品
情绪真实感：孩子的表情要自然真实，避免过度摆拍，让用户感受到真实的快乐氛围
合规性：如果是亚马逊平台，避免出现版权元素，背景的 “工地风” 装饰用通用款，不要出现品牌 logo
